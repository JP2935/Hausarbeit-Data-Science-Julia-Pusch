# Analyse der Corona-Rechtsprechung des BGH und des EuGH im Zeitraum 2020-2025

Dieses Repository enthält die Skripte zur quantitativen Analyse juristischer Urteile mittels Natural Language Processing (NLP).

## Projektübersicht
- **Datenbasis:** 216 Urteile (PDF)
- **Methodik:** Topic Modeling (LDA als Baseline, NMF als Hauptmodell)
- **Frameworks:** Python, SpaCy, Gensim, Scikit-Learn

## Inhalt der Skripte
- `03_LDA_Modellierung.ipynb`: Vorverarbeitung und Baseline-Modellierung.
- `04_NMF_Analyse.ipynb`: Optimierte Themenmodellierung und finale Visualisierungen.

## Ergebnisse
Die Analyse zeigt eine klare institutionelle Trennung zwischen BGH (Zivilrecht) und EuGH (Unionsrecht) sowie die zeitliche Evolution der Rechtsmaterien von 2020 bis 2025.
